# vdash-emu-templates

SLIMax Virtual Racing Display/Dash Emulator ( VDASH-EMU ) is a hardware Racing Display firmware emulator working on your Android tablet or phone or Windows PC/Laptop/Tablet. The EMUlator is powerful enough to display in real time the telemetry data and to give you a feedback on shiftlights and warning led like a real dash display available in racing cars. It contains already the emulation of 4 displays ready to use + 44 pages for your own custom displays and an integrated virtual ButtonBox to control your favorite game or simulation.
